# Lens

Lens is a single-file web app for analyzing potential political bias, emotional framing, credibility, and sourcing in news articles.

Features:
- Paste article text directly or paste a URL and fetch it through `https://corsproxy.io/?url=ENCODED_URL`
- Analyze content with the OpenAI API using `gpt-4o`
- View inline partisan and emotional highlights with hover explanations
- Review a premium-style report with a bias meter, emotional intensity, summaries, credibility grade, and sources
- Try a built-in demo article without an API key

Built with OpenAI Codex.
